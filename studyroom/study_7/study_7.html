회원가입

회원가입 폼으로 공부한 내용 복습하기
<div class="signup-form"> -> 회원가입 폼 상자(signup-form 클래스)
<h2>회원가입</h2> -> 회원가입 재목 표시
<form> -> 입력된 데이터를 서버에 보낼 수 있게 감싸주는 태그
...
<button type="submit">가입하기</button> -> 가입 버튼, 눌렀을 때 폼 전송
</form>
</div>


<label for="name">이름</label> -> 이름이라는 글자를 보여줌
<input type="text" id="name" name="name" required> -> 실제 입력하는 칸
-> 같은 방식으로 전화번호, 주소, 아이디, 비밀번호, 생일 등 입력칸이 있음!


.signup-form { -> signup-form 클래스
background-color: white; -> 박스를 흰색으로
padding: 30px; -> 안쪽 여백
border-radius: 10px; -> 박스 모서리를 둥글게
max-width: 400px; -> 최대 넓이 제한
margin: auto; -> 가운데 정렬
box-shadow: 0 0 10px rgba(0,0,0,0.1);} -> 그림자 효과


.signup-form input { ... } -> 모든 입력칸(비밀번호, 텍스트)
.signup-form button { ... } -> 가입버튼

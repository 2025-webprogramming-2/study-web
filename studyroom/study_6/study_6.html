로그인 컴포넌트 코드 분석하기

로그인

아이디를 잊어버리셨나요?

1. html

<div class="login-container">


->login-container 클래스 형성 후 로그인 박스 만들기
(div 태그는 구획을 나누는 레이아웃 역할로 사용)


<h2>로그인 </h2>
<img src="pencil.svg" alt="로고" class="title-logo">


-> 로그인이라는 큰제목을 표현
h2태그는 글자크기를 조절
img는 사진을 보여줄때 사용하며 "pencil.svg" 를 로고로 사용
("title with-logo"클래스)


<label for="username">사용자 이름</label>
<input type="text" id="username" name="username" required>


-> 로그인/패스워드 입력칸
label태그에 사용자 이름이라고 이름을 보여준다.
input 칸에 실제 아이디를 입력받는다. required는 꼭 입력해야 한다는 뜻
("input-group"클래스)


<input type="checkbox" id="remember" name="remember">
<label for="remember">아이디 기억하기</label>


-> 아이디 저장하기 체크박스
아이디/패스워드 입력칸과는 다르게 input 타입을 checkbox로 만들어서
체크할수있는 박스를 만듬("remember-me" 클래스)


<button type="submit" class="login-btn">공부하러 가기!</button>


->로그인 버튼 button태그를 이용해서 클릭했을 때 로그인시도


2. css
: Arial, sans-serif; -> 글꼴 설정
background-image: url('back.jpg'); -> 배경 이미지 가져오기
background-size: cover; -> 화면에 이미지 꽉 채우기
background-position: center; -> 배경화면 가운데 정렬
background-repeat: no-repeat; -> 배경화면 반복없이 한번만
display: flex; -> 가운데 정렬
justify-content: center; -> 가로 방향 가운데 정렬
align-items: center; -> 새로 방향 가운데 정렬
height: 100vh; -> 화면 높이 100&
margin: 0; -> 버깥 여백 엾애기



.login-container { ->login-container 클래스 로그인 박스 꾸미기
background-color: #e6f9e6; -> 연한 연두색 배경
padding: 30px; -> 안쪽 여백
border-radius: 8px; -> 모서리 둥글게
box-shadow: ... -> 그림자 효과
width: 100%;
max-width: 400px; -> 최대 너비 제한
text-align: center;} -> 글씨 가운데 정렬


.input-group { -> input-group 클래스(그룹 전체)
margin-bottom: 15px; -> 바닥에서 부터의 여백
text-align: left;} -> 왼쪽부터 텍스트 입력
.input-group label { } -> (라벨(아이디,비밀번호 글씨))
.input-group input { } -> (실제 입력창)
.input-group input:focus { } -> 입력칸을 클릭할 때 테두리 색을 바꿔줌


.title-with-logo {
-> title-with-logo 클래스 display: flex;
align-items: center;
justify-content: center;
gap: 15px;
margin-bottom: 20px;}

.title-with-logo h2 {
font-size: 24px;
color: #333;
margin: 0;}

.title-logo {
width: 28px;
height: 28px;}
-> 박스와 만드는 방법 동일!
